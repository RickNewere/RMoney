# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

This is the web/iPhone client of RMoney — a single static PWA (`index.html`, all CSS/JS inline, no build step) deployed to GitHub Pages at https://ricknewere.github.io/RMoney/ by `.github/workflows/deploy.yml` on every push to `main`. The repo must stay public: GitHub Free doesn't serve Pages from private repos.

Full system documentation (backend API, spreadsheet column logic, deployment gotchas) lives in the native-app project: `C:\Users\rober\Desktop\RMoney\CLAUDE.md`. Key points that apply here:

- The `API` constant and the `TABS` gid map at the top of `index.html` are duplicated in the native app's `config.js` — keep them in sync when the Apps Script URL or tabs change.
- POST uses `Content-Type: text/plain;charset=utf-8` on purpose (avoids CORS preflight with Apps Script) — don't switch to `application/json`.
- UI is Italian and must stay visually identical to the Android app (brand "💸 RMoney", same fields: Persona, Conto, Data, Importo, Categoria, Nota, checkbox that appends `-` right of the note column).
- No PIN/auth: considered and explicitly rejected by the owner.
- Icons (apple-touch-icon.png, icon-192/512.png, favicon.png) are generated by `RMoney\backend\make_icon.py` — regenerate there, don't hand-edit.
